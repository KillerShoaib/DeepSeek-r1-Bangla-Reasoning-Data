2b²*(√2/2) = 2b² - b²√2\n\n=> 200 = b² (2 - √2)\n\nসুতরাং b² = 200 / (2 - √2) ≈ 200 / (0.5858) ≈ 341.42 → b≈18.47 cm, যা আগের গণনার সাথে মেলে।\n\nদ্বিতীয় ক্ষেত্রে, কোণ A=135°, cos 135°= -√2/2।\n\nসুতরাং a² = 2b² - 2b² cos 135° = 2b² - 2b²*(-√2/2) = 2b² + b²√2\n\nসুতরাং:\n\n200 = 2b² + b²√2 = b²(2 + √2)\n\nঅতএব, b² = 200 / (2 + √2) ≈ 200 / 3.414 ≈ 58.58 → b≈7.65 cm।\n\nসুতরাং উভয় ক্ষেত্রেই সম্ভব। সমস্যাটিতে বলা হয়েছে পরিলিখিত বৃত্তের ব্যাসার্ধ 10 cm, যা আমরা ব্যবহার করেছি, তবে কোন কোণ A সঠিক তা কীভাবে সমাধান করব।\n\nএকটু দাঁড়ান, সম্ভবত সমস্যাটিতে বলা হয়েছে যে ত্রিভুজটি সমদ্বিবাহু, যা বাংলায় সাধারণত দুটি সমান বাহু বিশিষ্ট ত্রিভুজকে বোঝায়। \"সমদ্বিবাহু\" শব্দটি \"isosceles\" হিসাবে অনুবাদ করা যেতে পারে, যার দুটি সমান বাহু বা দুটি সমান কোণ থাকতে পারে। তবে যাই হোক না কেন, যদি বাহু b এবং c সমান হয়, তবে কোণ B এবং C সমান।\n\nএকটু দাঁড়ান, তবে সমস্যা বিবৃতিতে বলা হয়েছে \"b=c\", তাই বাহু b এবং c সমান। সুতরাং ত্রিভুজটি বাহু b এবং c সমান বিশিষ্ট সমদ্বিবাহু। কোণ A-এর দুটি সম্ভাবনা হলো 45° এবং 135°, যা কোণ B এবং C কে যথাক্রমে 67.5° এবং 22.5° তে নিয়ে যায়।\n\nতবে আমরা কীভাবে নির্ধারণ করব কোনটি সঠিক। হুমম।\n\nএকটু দাঁড়ান, সম্ভবত আমার পদ্ধতিতে একটি ভুল আছে। আসুন সমস্যাটি পুনরায় প্রকাশ করি। যেহেতু ত্রিভুজটি বাহু b = c বিশিষ্ট সমদ্বিবাহু, তাই কোণ B এবং C সমান। আসুন কোণ B এবং C উভয়কে θ ধরি। সুতরাং কোণ A = 180° - 2θ।\n\nসাইনের সূত্র ব্যবহার করে, আমরা পাই:\n\na / sin A = 2R → 10√2 / sin(180° - 2θ) = 2*10 → 10√2 / sin(2θ) = 20 → sin(2θ) = 10√2 /20 = √2/2।\n\nসুতরাং sin(2θ) = √2/2। 2θ এর সম্ভাব্য সমাধান হলো 45° বা 135°, যেহেতু এই কোণগুলোতে 0° এবং 180° এর মধ্যে সাইনের মান √2/2।\n\nঅতএব, 2θ = 45° → θ = 22.5°, অথবা 2θ = 135° → θ = 67.5°।\n\nএকটু দাঁড়ান, একটু দাঁড়ান, এটি ভিন্ন। পূর্বে, আমি কোণ A কে 45° বা 135° বিবেচনা করেছিলাম, তবে এখানে, সাইনের সূত্র ব্যবহার করে, আমি 2θ কে 45° বা 135° পাই, যা θ=22.5° বা 67.5° তে নিয়ে যায়। তবে এটি কীভাবে কোণ A এর সাথে সম্পর্কিত?\n\nকারণ কোণ A = 180° - 2θ। সুতরাং যদি 2θ =45° হয়, তবে কোণ A=180-45=135°, এবং যদি 2θ=135° হয়, তবে কোণ A=180-135=45°।\n\nসুতরাং এটি দুটি সম্ভাবনার দিকে নিয়ে যায়:\n\n1. যদি 2θ=45° হয়, তবে θ=22.5° (কোণ B এবং C প্রতিটি 22.5°), কোণ A=135°।\n\n2. যদি 2θ=135° হয়, তবে θ=67.5°, কোণ A=45°।\n\nউভয়ই বৈধ। সুতরাং কোণ B 22.5° বা 67.5° হতে পারে, যা অপশন D এবং B। তাহলে কেন দুটি সম্ভাব্য উত্তর রয়েছে? সমস্যাটির অবশ্যই একটি অনন্য সঠিক উত্তর থাকতে হবে। সম্ভবত সমস্যা বিবৃতিতে একটি ভুল আছে, অথবা সম্ভবত আমি কিছু মিস করছি।\n\nএকটু দাঁড়ান, আসুন অপশনগুলো আবার পরীক্ষা করি। অপশনগুলো হলো A)75°, B)67.5°, C)15°, D)22.5°। সুতরাং উভয় B এবং D দুটি পরিস্থিতির উপর ভিত্তি করে সম্ভব। তবে যেহেতু সমস্যাটি একটি বহু-নির্বাচনী প্রশ্ন যেখানে শুধুমাত্র একটি সঠিক উত্তর থাকতে পারে, তাই একটিকে অন্যটির চেয়ে বেছে নেওয়ার একটি কারণ থাকতে হবে।\n\nসম্ভবত বিভ্রান্তি বাহুগুলোর সংজ্ঞা থেকে আসে। ত্রিভুজ চিহ্নাঙ্কনে, বাহু a কোণ A-এর বিপরীত। সমস্যাটিতে বলা হয়েছে যে a=10√2 cm। যদি কোণ A 135° হয়, তবে বাহু b এবং c প্রতিটি 7.65 cm, তবে যদি কোণ A 45° হয়, তবে বাহু b এবং c প্রতিটি 18.47 cm। সমস্যাটিতে অন্য কোনো শর্ত উল্লেখ করা নেই, তাই উভয়ই সম্ভব, তবে উত্তর অপশনগুলোতে B এবং D উভয়ই অন্তর্ভুক্ত।\n\nহুমম। এটি ইঙ্গিত করে যে সমস্যাটিতে একটি টাইপো বা অস্পষ্টতা থাকতে পারে। তবে যেহেতু এটি একটি বহু-নির্বাচনী প্রশ্ন, তাই সম্ভবত আমাদের অন্য একটি পদ্ধতি বিবেচনা করতে হবে।\n\nএকটু দাঁড়ান, ত্রিভুজের পরিলিখিত বৃত্তের ব্যাসার্ধের অন্য একটি সূত্র হলো R = (a b c)/(4 area)। আসুন এটি চেষ্টা করি।\n\nযদি কোণ A 45° হয়, তবে ক্ষেত্রফল = (1/2) * b * c * sin A। যেহেতু b = c, ক্ষেত্রফল = (1/2) * b² * sin 45°।\n\nকিন্তু a =10√2, এবং আগে থেকে, b=18.47 cm। সুতরাং ক্ষেত্রফল = 0.5 * (18.47)^2 * (√2/2) ≈ 0.5 * 341 * 0.707 ≈ 0.5 * 341 * 0.707 ≈ 120.7 cm²।\n\nতাহলে R = (a b c)/(4 area) = (10√2 * 18.47 * 18.47)/(4*120.7)।\n\nলব গণনা করা হচ্ছে: 10√2 ≈14.14, 14.14 * 18.47^2 ≈14.14 * 341 ≈4823.74।\n\nহর: 4*120.7≈482.8।\n\nসুতরাং R≈4823.74 / 482.8 ≈10। যা মিলে যায়।\n\nঅন্য ক্ষেত্রে, কোণ A=135°, ক্ষেত্রফল = (1/2)*b²*sin 135° = 0.5*b²*(√2/2)=0.25*b²*√2।\n\nb≈7.65 cm হলে, b²≈58.5। সুতরাং ক্ষেত্রফল≈0.25*58.5*1.414≈0.25*58.5*1.414≈20.7 cm²।\n\nতাহলে R = (10√2 *7.65*7.65)/(4*20.7) ≈ (14.14 *58.5)/(82.8) ≈827.19/82.8≈10, যা মিলে যায়। সুতরাং উভয় ক্ষেত্রেই R=10 cm সন্তুষ্ট হয়।\n\nঅতএব, কোণ B=67.5° এবং 22.5° উভয়ই সম্ভব, তবে অপশনগুলোতে উভয়ই অন্তর্ভুক্ত। তাহলে কেন একটি সঠিক?\n\nএকটু দাঁড়ান, সম্ভবত সমস্যাটিতে বলা হয়েছে যে ত্রিভুজটি \"সমদ্বিবাহু\" যা কিছু ক্ষেত্রে ভূমি সংলগ্ন সমান বাহুগুলোকে উল্লেখ করতে পারে। তবে এটি প্রয়োজনীয়ভাবে একটি আদর্শ সংজ্ঞা নয়। বিকল্পভাবে, সম্ভবত সমস্যাটির চিহ্নাঙ্কনে একটি প্রথা রয়েছে যে বাহু a হলো ভূমি, তাই যদি ত্রিভুজটি বাহু b এবং c সমান বিশিষ্ট সমদ্বিবাহু হয়, তবে ভূমি হলো a, এবং দুটি সমান বাহু হলো b এবং c। যদি ত্রিভুজটির ভূমি a হয়, তবে ভূমির কোণগুলো হলো B এবং C। একটু দাঁড়ান, তবে আদর্শ চিহ্নাঙ্কনে, কোণ A বাহু a-এর বিপরীত। সুতরাং যদি a ভূমি হয়, তবে কোণ B এবং C ভূমিতে থাকবে এবং সমান হবে। তবে আমাদের আগের ক্ষেত্রে, যদি কোণ A 45° হয়, তবে কোণ B এবং C হলো 67.5°, যা ভূমির কোণ। যদি কোণ A 135° হয়, তবে ভূমির কোণ হলো 22.5°, যা সম্ভব।\n\nতবে সম্ভবত সমস্যাটিতে একটি ভুল আছে, অথবা সম্ভবত আমাকে অন্য একটি পদ্ধতি বিবেচনা করতে হবে। বিকল্পভাবে, সম্ভবত উত্তরটি 67.5°, যা অপশন B, কারণ যদি কোণ A 45° হয়, তবে ত্রিভুজটির সমান বাহুগুলো দীর্ঘ হবে, যা সম্ভবত বেশি সাধারণ। তবে এটি কেবল একটি অনুমান। তবে যেহেতু অপশনগুলোতে উভয়ই অন্তর্ভুক্ত, সম্ভবত একটি ভুল গণনা রয়েছে।\n\nএকটু দাঁড়ান, অন্য একটি পদ্ধতি: বাহু এবং ক্ষেত্রফলের পরিপ্রেক্ষিতে পরিলিখিত বৃত্তের ব্যাসার্ধের সূত্র ব্যবহার করে। তবে এটি সাহায্য নাও করতে পারে। বিকল্পভাবে, সম্ভবত R = a/(2 sin A) সূত্র ব্যবহার করে এবং কোণ B কে কোণ A এর পরিপ্রেক্ষিতে প্রকাশ করে।\n\nএকটু দাঁড়ান, যেহেতু কোণ B কোণ C এর সমান, এবং কোণ A = 180 - 2B, আমরা লিখতে পারি:\n\nR = a/(2 sin A) = a/(2 sin(180 - 2B)) = a/(2 sin 2B)\n\nকারণ sin(180 - x) = sin x।\n\nসুতরাং R = a/(2 sin 2B)\n\nদেওয়া আছে R=10 cm, a=10√2 135°, তাই B=22.5° বা 67.5°।\n\nআবার, দুটি সম্ভাব্য উত্তর। তবে অপশনগুলোতে B এবং D উভয়ই রয়েছে। সুতরাং সম্ভবত প্রশ্নটি আশা করে যে আমরা বিবেচনা করব যে b=c বাহু বিশিষ্ট একটি ত্রিভুজে, কোণ B এবং C অবশ্যই
